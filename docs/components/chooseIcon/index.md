## 图标选择器

用户点击按钮, 弹出框显示所有图标, 点击图标可以复制图标代码

### 按需引入

```js
import chooseIcon from "n-element-components/chooseIcon";
import "n-element-components/chooseIcon/style.css";

// main.ts
app.use(chooseIcon);
```

### 基本用法

<br>
